TigerScale Web Installer

Flash your TigerScale firmware directly from the browser. No Arduino IDE, no PlatformIO, no command line.

⚠️ Browser not supported ESP Web Tools requires the Web Serial API, which only works in Chrome, Edge, Brave, or Opera. Please open this page in one of those.

Install in 60 seconds

1
Plug in your ESP32 via USB Use a data cable (not charge-only). The ESP32 should be visible to your computer.
2
Click "Install TigerScale" below A popup will ask which serial port to use — pick your ESP32 (typically cu.usbserial-XXXX on macOS, COMx on Windows, ttyUSB0 on Linux).
3
Wait ~60 seconds The browser flashes the firmware and the web UI. Don't unplug while it runs.
4
Configure Wi-Fi After install, the scale boots into setup mode. Connect your phone to the Setup-TigerScale-XXXX Wi-Fi network and follow the captive portal.

Having trouble? See TROUBLESHOOTING.md

What gets installed?

  • Firmware (~1.3 MB) — the C++ code that runs on the ESP32
  • Web interface (~900 KB) — the page you'll see at tigerscale-XXXX.local
  • Bootloader + partition table

Your existing settings (Wi-Fi, Firebase login, calibration) will be preserved.

Don't trust the browser to flash your device?

The TigerScale firmware is fully open-source. You can :

  • Read every line of code on GitHub
  • Compile and flash it yourself with PlatformIO or Arduino IDE
  • Verify the SHA256 of the published binary against your own build

The Web Installer is just a convenience — it uses the Web Serial API which is sandboxed by your browser : the page only sees the device when you explicitly grant access via the popup.
